# Machine Learning  

This repository contains a series of practical labs focused on fundamental machine learning concepts. The projects cover the full workflow of a machine learning problem, from data exploration and preprocessing to model training, evaluation, and improvement. Different types of models are explored, including regression, classification, ensemble methods, and neural networks.

===========================================================================

## Practicum 1: Introduction to Machine Learning
===========================================================================

The main topics are:

- Data exploration and visualization
- Train/validation/test split
- Basic regression models
- Overfitting

This practicum introduces the basic workflow of a machine learning project. It includes exploring numerical and categorical data, splitting datasets, and training a simple regression model using default parameters.  

The second part focuses on completing a full ML pipeline: training, evaluating, and slightly improving a simple model (Linear Regression or KNN). The main goal is to understand overfitting and how data preprocessing can improve results.

===========================================================================

## Practicum 2: Support Vector Machines and Imbalanced Data
===========================================================================

The main topics are:

- Imbalanced datasets
- Evaluation metrics
- Support Vector Machines (SVM)

In this practicum, different strategies to handle imbalanced data are explored, along with understanding which evaluation metrics are appropriate for each problem.  

The project focuses on using SVM models (linear and kernel versions), studying their parameters, and tuning them properly to improve performance.

===========================================================================

## Practicum 3: Boosting and Decision Trees
===========================================================================

The main topics are:

- Decision trees
- Ensemble methods (boosting)
- Custom model implementation

This practicum focuses on understanding how decision trees work and how ensemble techniques can improve simple models.  

A boosting classifier is implemented from scratch using decision trees as weak learners. The project also includes implementing and using Weighted Binary Cross Entropy as a loss function.

===========================================================================

## Practicum 4: Convolutional Neural Networks (CNN)
===========================================================================

The main topics are:

- Deep learning
- CNN architecture design
- Image data processing

In this practicum, a convolutional neural network is trained to count object occurrences in images.  

The project covers key concepts such as designing the network architecture, choosing activation functions, understanding filters, and handling issues like overfitting and vanishing gradients. Techniques like data augmentation and model simplification are also explored.
